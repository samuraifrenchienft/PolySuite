# Trend Scanner Alert Format

## Current Output
- Just sends token name/symbol
- Needs more detail

## Proposed Alert Format

```
🚨 TREND ALERT - pump.fun

📛 Name: [token name]
💰 Symbol: [token]
📝 Desc: [description - 50 chars]
🔗 Link: pump.fun/[symbol]

🤖 AI Analysis:
[AI reason - 1 sentence]
```

## Fields to Include
- Name, Symbol
- Description (truncated)
- Mint address
- Twitter if available
- AI analysis

## Market Alerts Format
Similar structure for:
- New markets
- Arbitrage
- Convergence
- Volume spikes
