# Wallet Tracking System - SIMPLE

## User Experience

Users find wallets however they want (Twitter, leaderboards, friends, etc.)

Then simply add them:
- `/add 0xABC...` - Add wallet to track
- `/remove 0xABC...` - Stop tracking
- `/status` - See what you're tracking

## Limits
- Max **100 wallets** per user
- Reason: Enough for serious tracking, not enough to abuse

## Whale Alerts
When any tracked wallet makes a trade → Alert!

That's it. Simple.
